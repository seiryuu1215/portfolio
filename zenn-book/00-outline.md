# Claude Codeで55,000行のWebアプリを3ヶ月で作った全記録

## 本の概要

プロダーツ選手が、自身の課題を解決するためにClaude Codeと協働してフルスタックWebアプリ「darts Lab」を企画〜運用まで一人で構築した全記録。
AI駆動開発のリアルな実践記録として、何をAIに任せ、何を自分で判断したかを具体的に記録する。

**想定読者**: AI駆動開発に興味のあるエンジニア / 個人開発者 / フリーランス
**プラットフォーム**: Zenn（本）
